HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is a series of elements that make different parts of the content appear or act a certain way.
  • An element usually includes an opening tag with an attribute, text content, and closing tag.
  • An attribute is used in an element to contain more information about how you want the content to appear.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector is the HTML element picked to style.
  • Different properties can be used to style elements in various ways.
  • An ID selector modifies a specific element on the page, whereas a CLASS selector modifies a group of elements on the page.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: updates merges remote repo into local repo
  • git push origin "branchname": pushes branch from local repo to remote repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditionals determine if a block of code is used, depending on whether or not a specific condition is met.
  • An array is a single variable that holds a group of data.
  • Functions are statements that perform a task or calculate a value.
